# 有二品牌内容小程序 - 项目交接文档

> **文档生成时间**：2026-04-11
> **项目名称**：有二内容小程序
> **技术栈**：Taro 4 + NestJS + React 18 + TailwindCSS 4

## 一、项目概述

### 1.1 项目定位
为"有二财经"和"有二会客"两个品牌账号开发的小程序，用于展示每日AI相关资讯速览和朋友圈文案库。

### 1.2 核心功能
- **有二财经**：展示AI财税、AI企业落地等9个主题的每日资讯速览
- **有二会客**：展示AI改变个体、一人公司变现等9个主题的每日资讯速览
- **文案库**：朋友圈文案展示和一键复制功能

### 1.3 品牌视觉
- **有二财经**：冰川浅蓝背景（#E8F4FD）、深藏青文字（#1B3A6B）、专业冷静风格
- **有二会客**：深靛蓝背景（#3B4C7D）、白色文字、沉稳理性风格

## 四、快速开始

### 4.1 环境要求
- Node.js >= 18
- pnpm >= 9.0.0

### 4.2 安装依赖
```bash
cd /workspace/projects
pnpm install
```

### 4.3 启动开发服务器
```bash
# 启动前端+后端（推荐）
pnpm dev

# 或分别启动
pnpm dev:web   # H5开发服务器（端口5000）
pnpm dev:server # NestJS后端（端口3000）
```

### 4.4 构建项目
```bash
# 构建所有平台
pnpm build

# 构建微信小程序
pnpm build:weapp

# 构建抖音小程序
pnpm build:tt

# 构建H5
pnpm build:web
```

### 4.5 代码校验
```bash
# 运行ESLint和TypeScript检查
pnpm validate

# 仅运行ESLint
pnpm lint:build

# 仅运行TypeScript
pnpm tsc
```

## 七、后端API说明

### 7.1 已开发但未使用的接口
**问卷模块**（server/src/survey/）

- `POST /api/survey/submit` - 提交问卷
- `GET /api/survey/results/:surveyId` - 获取问卷结果

### 7.2 启动后端
```bash
cd /workspace/projects
pnpm dev:server
```

后端默认运行在 `http://localhost:3000`

### 7.3 网络请求规范
**前端使用**：
```typescript
import { Network } from '@/network';

// GET请求
const data = await Network.request({ url: '/api/health' });

// POST请求
const result = await Network.request({
  url: '/api/survey/submit',
  method: 'POST',
  data: { surveyId: 1, answers: {} }
});
```

**重要规则**：
- ✅ 使用相对路径 `/api/xxx`
- ❌ 禁止硬编码 `http://localhost:3000`
- ❌ 禁止直接使用 `Taro.request`

## 九、注意事项

### 9.1 代码规范
- **强制使用pnpm**：禁止使用npm或yarn
- **代码校验**：提交前必须运行 `pnpm validate`
- **组件使用**：优先使用 `src/components/ui` 下的组件，不要用 `View/Text` 手搓通用UI
- **样式规范**：优先使用TailwindCSS类名，避免硬编码px值

### 9.2 小程序限制
- **图片资源**：必须使用TOS对象存储，禁止打包到项目（TabBar图标除外）
- **包体积**：注意小程序包体积限制（2MB）
- **跨端兼容**：注意H5和小程序的兼容性差异

### 9.3 配置文件
- `project.config.json`：微信小程序配置
  - `appid`：需要配置真实的小程序AppID
  - `packOptions.ignore`：排除assets目录（避免包体积过大）

### 9.4 当前数据
- **数据来源**：目前使用模拟数据（硬编码在组件中）
- **未来对接**：需要对接真实的每日推送数据源（Notion/数据库/API）
