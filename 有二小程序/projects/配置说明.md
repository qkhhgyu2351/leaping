# 如何配置小程序 AppID 以生成预览二维码

## 问题原因
当前 project.config.json 中的 appid 是测试用的 `touristappid`，需要配置您已备案的真实小程序 AppID。

## 解决方法

### 方法一：通过 Coze 预览区配置（推荐）
1. 找到右侧的 **预览区**
2. 点击预览区中的 **配置按钮**
3. 在弹出的配置窗口中，填写您的 **微信小程序 AppID**
4. 保存配置

### 方法二：手动修改配置文件
如果您知道自己的 AppID，也可以直接修改：

**文件路径**: `/workspace/projects/project.config.json`

**需要修改的内容**:
```json
{
  "appid": "您的真实AppID"  // 把这里的 "touristappid" 替换为您的真实 AppID
}
```

**示例**:
```json
{
  "appid": "wx1234567890abcdef"
}
```

修改后需要重新构建：
```bash
cd /workspace/projects && pnpm build:weapp
```

## 如何获取 AppID
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入您的已备案小程序
3. 在 **开发 > 开发管理 > 开发设置** 中找到 AppID（以 wx 开头）

## 配置完成后
- Coze 平台会自动生成预览二维码
- 您可以直接在右侧预览区看到二维码
- 使用微信扫码即可预览小程序
