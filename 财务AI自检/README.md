# 增重教练项目

本项目包含一个智能的个性化增重教练应用，帮助用户实现科学增重目标。

## 项目内容

- **bulking-app/** - 主要应用目录
  - `bulking-coach.html` - 个性化增重教练应用（推荐使用）
  - `bulking-tracker.html` - 基础增重追踪工具
  - `README.md` - 详细的项目文档

## 如何使用

1. 进入 `bulking-app` 文件夹
2. 打开 `bulking-coach.html` 文件开始使用
3. 完成入职流程，创建个性化增重计划
4. 开始记录每日体重和饮食

## 核心功能

- 个性化增重计划生成
- 科学的热量和蛋白质目标计算
- 分阶段增重策略
- 智能建议系统
- 数据可视化图表
- PWA支持，可添加到主屏幕

## 技术特点

- 基于BMR和TDEE的科学算法
- 响应式设计，支持所有设备
- 本地数据存储，完全隐私
- 现代化UI界面
- 实时数据可视化
