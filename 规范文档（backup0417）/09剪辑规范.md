# 剪辑规范

> 本文档为「有二财经」与「有二会客」视频剪辑的统一规范。
> 财经号和会客号有各自规则，通用规则共用。

---

## 一、字幕规范

### 1.1 通用规则

| 项目 | 规范 |
|------|------|
| 字体 | 思源黑体 / 阿里巴巴普惠体（不用宋体、楷体） |
| 字号 | 主字幕 48-56px，重点词可放大到 64-72px |
| 位置 | 底部居中，距离底部边缘 80-100px |
| 颜色 | 白色为主，重点词用强调色 |
| 描边 | 黑色描边 2-3px，确保在浅色背景上可读 |
| 动效 | 逐字出现或整句淡入，不用弹跳、旋转等花哨动效 |

### 1.2 财经号差异

| 项目 | 规范 |
|------|------|
| 强调色 | 警示橙 #F97316（用于数字、关键结论） |
| 数字处理 | 百分比、金额用等宽字体，对齐显示 |
| 风格 | 简洁、专业、无装饰 |

### 1.3 会客号差异

| 项目 | 规范 |
|------|------|
| 强调色 | 暖琥珀 #F5A623（用于金句、关键转折） |
| 氛围 | 可以加轻微的呼吸动效，但不花哨 |
| 风格 | 温暖、有人味、像朋友聊天 |

## 二、剪辑节奏

### 2.1 通用规则

| 项目 | 规范 |
|------|------|
| 时长 | 口播 3-4 分钟，不超过 5 分钟 |
| 切换频率 | 每 5-8 秒一个画面变化（字幕、镜头、图片） |
| 留白 | 句与句之间留 0.5-1 秒呼吸空间 |
| 开头 | 前 3 秒必须有钩子，不能有废话开场 |
| 结尾 | 最后 30 秒内出 CTA，不要拖沓 |

### 2.2 财经号差异

| 项目 | 规范 |
|------|------|
| 节奏 | 稳、准、狠，不拖泥带水 |
| 画面 | 以数据图、案例截图、文字卡片为主 |
| BGM | 不用或用极轻的背景音，不抢人声 |
| 风格 | 实在、精准，不飘在空中。像干了17年的内行在讲真的，有话直说，不端着 |

### 2.3 会客号差异

| 项目 | 规范 |
|------|------|
| 节奏 | 可以稍慢，给观众思考空间 |
| 画面 | 可以穿插个人照片、场景图、表情包（适度） |
| BGM | 可以用轻快的背景音乐，营造聊天氛围 |
| 风格 | 像朋友在咖啡馆聊天，自然、有温度 |

## 三、发布规范

### 3.1 通用规则

| 平台 | 封面比例 | 时长限制 | 标签数量 |
|------|----------|----------|----------|
| 小红书 | 3:4 竖版 | 不限 | 8-10 个 |
| 抖音 | 9:16 竖版 | 不限 | 5-8 个 |
| 视频号 | 9:16 竖版 | 不限 | 5-8 个 |

**发布时间策略：**
按内容类型匹配发布节奏：
- **税务合规/财务专业类**：工作日中午 12:00-13:00（财务从业者午休刷手机）
- **财报分析/企业AI落地类**：工作日晚上 20:00-21:30（老板/CFO晚上看决策参考）
- **OPC实验/转型故事/AI工具类**：周末晚上 20:00-22:00（转型者周末有空刷内容）
- **黄金原则**：你什么时候有空拍，就什么时候发——稳定更新比掐点更重要

### 3.2 财经号差异

| 项目 | 规范 |
|------|------|
| 封面风格 | 专业、数据感、警示橙强调 |
| 标题公式 | 具体数字 + 冲突点 + 企业视角 |
| 必带标签 | #有二财经 #AI多懂企业企业就有多强 |
| 评论区 | 置顶引导到财务 AI 准备度自测 |

### 3.3 会客号差异

| 项目 | 规范 |
|------|------|
| 封面风格 | 温暖、有人味、暖琥珀强调 |
| 标题公式 | 真实经历 + 转折 + 个人视角 |
| 必带标签 | #有二会客 #AI多懂你你就有多强 |
| 评论区 | 置顶引导到陪伴群 |

## 四、技术参数

### 4.1 通用规则

| 项目 | 规范 |
|------|------|
| 分辨率 | 1080p（1920×1080）或 720p（1280×720） |
| 码率 | 8-12 Mbps |
| 帧率 | 30fps |
| 格式 | MP4（H.264 编码） |
| 音频 | AAC，采样率 44.1kHz，码率 128-192kbps |

### 4.2 平台差异

| 平台 | 推荐分辨率 | 最大文件大小 |
|------|------------|--------------|
| 小红书 | 1080×1440（3:4） | 5GB |
| 抖音 | 1080×1920（9:16） | 4GB |
| 视频号 | 1080×1920（9:16） | 1GB |

## 五、检查清单

**剪辑完成后必检：**

- [ ] 前 3 秒有钩子
- [ ] 字幕无错别字、无漏字
- [ ] 数字、金额显示正确
- [ ] 音量适中，人声清晰
- [ ] 时长在 3-5 分钟内
- [ ] 结尾 CTA 清晰
- [ ] 封面图符合规范
- [ ] 标签包含必带标签
- [ ] 图片无元数据（期数/日期/版本号/文件编号等）
