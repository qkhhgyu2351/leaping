## 图片生成规范

**触发：** 根据文案，生成以下配图。

**输出要求：**
1.朋友圈1:1配图 × 1，结构：主标题 + 中间3D几何体图标 + 底部三段式
- 顶部：主标题，2-3行，每行≤12字，强调色（警示橙/增长绿）标注关键词
- 中间：3D几何体图标（立方体、球体、金字塔、环形等），横向排列，下方有简短标签文字
- 底部三段式（占图片高度15-20%，背景浅蓝色，无分隔线），底部对齐
  - 左1/4 = 二维码虚线框（纯框，框内无任何文字）
  - 中1/2 = `根据文案内容提炼7-12个字，两行显示`（纯文字，无背景，无箭头装饰）
  - 右1/4 = `有二会客 | 用AI的搞钱人`（纯文字，无背景，无色块）

2.公众号封面 × 1（2.35:1），结构：大标题 + 封面插图（3D几何体）
3.公众号正文配图 × 3（2.35:1），结构：无大标题，纯内容插图（3D几何体），对应文章中某个具体段落的核心要点，每张图只服务一个段落。
4.竖屏视频封面 × 1（抖音/视频号/，9:16），结构：竖版标题图，大标题 + 视频封面插图（3D几何体）
  底部中间加上：`有二会客 | 用AI的搞钱人`，占图片高度约10-12%，底部以上20-30px处居中，品牌文字字号12-14px，颜色跟随各账号主色或白色
5.小红书封面 × 1（3:4竖版），结构：封面图，大标题 + 封面插图（3D几何体）
  底部中间加上：`有二会客 | 用AI的搞钱人`，占图片高度约10-12%，底部以上20-30px处居中，品牌文字字号12-14px，颜色跟随各账号主色或白色

**图片标题：** 尽量使用文案原文中的核心标题或核心句子，也可自行创作，但标题必须表达文案核心意思，也必须有钩子吸引力，读标题就能判断文案在讲什么。
**通用禁止项：** 仅朋友圈1:1配图有底部三段式。其余图种禁止出现二维码、引导语；竖屏9:16和小红书3:4底部居中品牌文字除外（品牌文字规格见各图种输出要求）。
**元数据禁止项：** 图片中禁止出现期数、日期、版本号、文件编号等元数据。

## 视觉风格

### 色值（锁定）
| 元素 | 色值 |
|------|------|
| 主背景 | `#3B4C7D`（深靛蓝） |
| 内容卡片 | `#FFFFFF`（纯白） |
| 标题文字 | `#FFFFFF`（纯白） |
| 强调点缀 | `#F5A623`（暖琥珀） |
| 次级信息 | `#6B7EC9`（浅靛蓝） |
| 警示/痛点 | `#FF6B35`（警示橙） |
| 增长/收益 | `#00C49A`（增长绿） |
| 底部三段背景 | `#E8F4FD`（冰川浅蓝） |

### 字体
- 标题：思源黑体 Bold，35-45px
- 正文：思源黑体 Regular，18-22px
- 标签文字：思源黑体 Medium，14-18px
- 品牌文字：思源黑体 Regular，12-14px

### 插图规则

**元素选择：** 3D几何体（立方体、球体、金字塔、环形等）组合，有阴影和立体感，保持抽象风格，不是写实3D。
- 插图必须与文案核心信息相关（读标题就能判断文案在讲什么）
- 风格：几何抽象矢量风，线条干净，有立体感但不厚重，有科技感但不冰冷

### 整体气质

深靛蓝为底，白色为主，暖琥珀点缀。
沉稳、理性、有洞察力。像一个理性的朋友在分享真实的经验，不是老师在教导。
禁止任何形式的微商感、鸡汤感、成功学感。
